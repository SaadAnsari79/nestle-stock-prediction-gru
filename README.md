# nestle-stock-prediction-gru
# Nestle Stock Price Prediction using GRU

##  Overview

This project predicts stock prices of Nestle India using a GRU (Gated Recurrent Unit) deep learning model.

##  Dataset

* Historical stock price data
* Features: Open, High, Low, Close, Volume

##  Technologies

* Python
* Pandas, NumPy
* PyTorch
* Matplotlib

##  Model

* GRU (Recurrent Neural Network)
* Used for time series forecasting

##  Results

The model learns trends in stock data and predicts future values with reasonable accuracy.

##  Conclusion

GRU works well for sequential data like stock prices. Performance can be improved using LSTM or tuning.
